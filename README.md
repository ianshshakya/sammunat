# Sammunat Landing Page

A modern, responsive, glassmorphic landing page built using **React.js** and **Tailwind CSS** as part of the internship assignment for **Sammunat (USA)**.

This project demonstrates clean UI/UX, responsive design, modern frontend practices, and production-ready deployment.
